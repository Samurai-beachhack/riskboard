# Usage

---

## Basic Usage

### Analyze the current directory:

```bash
./zerohour analyze
```

### Analyze a Specific Directory

```bash
./zerohour analyze -C path/to/project
```

### Output Modes
Plain output (no boxed UI):

```bash
./zerohour analyze --no-box
```

### Common Alternatives

```bash
npm run analyze
npx zerohour analyze
npm link && zerohour analyze
```
